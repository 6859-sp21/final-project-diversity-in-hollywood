Oscars Dataset

The data for all Oscars winners were retrieved from data.world.
The race for each individual was scraped from nndb.com.

Golden Globes Dataset

The data for all Oscars winners were retrieved from kaggle.com.
The race for each individual was scraped from nndb.com.

Screen Actors Guild Dataset

The data for all Oscars winners were retrieved from kaggle.com.
The race for each individual was scraped from nndb.com.

Top Grossing Movies Dataset

The data for the top 1000 grossing movies by adjusted gross revenue were retrieved from boxofficemojo.com.
Additional columns of top-billed actors, IMDb rating, and genre were sourced from the IMDb dataset on kaggle.com.
Cast race percentages for each race (White, Black, Asian, Hispanic, Other) were calculated based on the race scraped for each top-billed actor from nndb.com.

Movie Posters

Movie posters were retrieved using the TMDB API.

Industry White Actor Percentage

For the metrics on the home page of industry-wide white actor percentage, the data was retrieved from data.usa.io.
